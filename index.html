    before connecting

    1. TouchOSC / TouchOSC Bridge cannot provide the grid connection for this page. This is direct Web Serial: the grid must be plugged into this computer and selected from Connect grid.
    2. Use desktop Chrome or Edge. Safari, Firefox, and iPhone/iPad browsers do not expose Web Serial for this.
    3. If serialosc is holding the grid port, stop it first with brew services stop serialosc.
    4. After using this page, restart serialosc with brew services start serialosc if you want normal monome apps to see the grid again.
    5. For offline/local development, run python3 -m http.server 8000 from this folder and use http://localhost:8000/.
